Gestionador de tareas

puede crear tareas, marcar dichas tareas como hechas (por defecto estan en "no hechas") una vez que se cumplen, y ver todas las tareas creadas

en caso de que se quiera crear una tarea con un nombre que ya esta almacenado se le pedira otro, y si se quiere marcar como hecha una tarea que no existe se le pedira otro, lo mismo si no se pone ningun nombre.
